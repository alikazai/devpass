## Ways to Contribute

- Reporting bugs
- Suggesting features
- Implementing features
- Reviewing security
- Improving documentation
- UX / UI improvements

---

## Project Scope

DevPass is **not**:
- A general password manager
- A cloud SaaS
- A full secrets manager

---

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Submit a pull request

```bash
git checkout -b feature/my-feature
```

---

## Development Rules

- TypeScript preferred
- No custom crypto
- Manifest V3 only
- Minimal permissions

---

## Reporting Security Issues

Do not open public issues.

Use a private GitHub security advisory.
